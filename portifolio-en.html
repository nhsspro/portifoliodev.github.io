Projects

Pre-order E-commerce System

I developed a web-based e-commerce system for Âtlantica, a company specializing in luxury pet furniture, as part of my Final Project for the Analysis and Systems Development course at ETEC Jardim Ângela. As a full-stack developer, I created a pre-order platform with features for product management, inventory control, and order processing, using technologies like HTML, CSS, JavaScript, PHP, and MySQL. The system was designed to be intuitive, secure, and scalable, providing a better shopping experience for customers and streamlining business administration.

Brasão Steakhouse

I developed the website for Brasão Steakhouse, a fictional steakhouse that offers a high-quality gastronomic experience. The website was created with a responsive design, ensuring accessibility across different devices. Using HTML, CSS, and JavaScript, I aimed to create a clean and intuitive interface, highlighting the menu, location, hours, and photo gallery. This project enhanced my skills in web development, design, and usability, reflecting the quality and tradition of the steakhouse.
