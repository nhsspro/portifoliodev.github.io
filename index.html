Projetos

Sistema e-commerce de pré-vendas

Desenvolvi um sistema web de e-commerce para a empresa Âtlantica, especializada em móveis de luxo para pets, como parte do meu Projeto de Conclusão de Curso em Análise e Desenvolvimento de Sistemas na ETEC Jardim Ângela. Como desenvolvedor full-stack, criei uma plataforma de pré-vendas com funcionalidades de gestão de produtos, controle de estoque e processamento de pedidos, utilizando tecnologias como HTML, CSS, JavaScript, PHP e MySQL. O sistema foi projetado para ser intuitivo, seguro e escalável, proporcionando uma melhor experiência de compra para os clientes e facilitando a administração da empresa.

Churrascaria Brasão

Desenvolvi o site para a Churrascaria Brasão, uma churrascaria fictícia que oferece uma experiência gastronômica de qualidade. O site foi criado com design responsivo, garantindo acessibilidade em diversos dispositivos. Utilizando HTML, CSS e JavaScript, busquei criar uma interface limpa e intuitiva, destacando o cardápio, localização, horários e galeria de fotos. Este projeto aprimorou minhas habilidades em desenvolvimento web, design e usabilidade, refletindo a qualidade e tradição da churrascaria.
